Create your own portfolio website using React.js

Raghavi Adoni

Oct 26, 2019·5 min read

How about having your own “About-Me” website or your own portfolio which would be on the web? Follow along and by the end of it, you’ll have your own portfolio built using React.js.

What you should know before you start :

The article assumes that you’ve a basic understanding of JavaScript, HTML and CSS. I highly recommend covering these basics well as it’ll help you understand better.

What exactly is React.js ?

React.js is a JavaScript library created by Facebook. It is widely used by developers to build User Interfaces (UIs).

Setting up React :

Follow the steps below to install React.js:

  • You’ll first need to install NodeJS which is JavaScript runtime environment and will help compile cool React code. Follow this link to install NodeJS.
  • Next, we’ll be installing react using the create-react-app command. Open the console and run npm install create-react-app command.
  • Now run npm create-react-app ${your-project-name} (for example: npm create-react-app about-me) command on the console directly or inside the console of any IDE ( I’ll be using Visual Studio) . This will create a folder of your project name with all the required files.
  • Then run cd ${your-project-name} , followed by npm start command and if everything has been installed perfectly till now, you’ll see your project running on the localhost.

Let’s Begin….

It’s important to understand the concept of ‘Components’ in React as they are the building blocks of any React app. UI can be broken down into components which are independent and reusable blocks of code in React.

Components can be of 2 types:

  • Functional Component which is declared as given below:
function Component_example() {
  return <div>Hi,I am a Component!</div>;
}
export default Component_example;

‘export default’ helps to export the component so that it can be imported and used anywhere in the project.

  • Class Component which is declared as given below:
class Component_example extends React.Component {
  render() {
    return <div>Hi,I am a Component!</div>;
  }
}
export default Component_example;

Class Components have some additional features which this article won’t cover as those features are not needed right now.

Anything that you write after the ‘return’ keyword is what your component will display at the front-end. Now that we know what components are, identify such independent parts of the UI and they will be your components.

All the code of our application will reside in the src folder. In the index.js file, the line ReactDOM.render(<App />, document.getElementById(‘root’)); basically places our ‘App’ component inside the <div id="root"></div> defined in the index.html which is located in the public folder. By doing so, all the contents of our App component are displayed on the webpage. We will thus replace the existing code inside the App component with our Header, Footer and the Tab components so that we’ll be able to view them.

Given below is the code of the App component.

import React from 'react';
import logo from './logo.svg';
import './App.css';
import Header from './components/Header';
import HomeTabs from './components/HomeTabs';
import Footer from './components/Footer';function App() {
  return (
    <div>
    <div className="app">
      <header className="app-header">
        <img src={logo} className="app-logo" alt="logo" 
          />
//placing the header component
      <Header />
      </header> 
//placing the tabs component
      <HomeTabs />
      </div>
    <div className='icons-List'>
//placing the footer component  
     <Footer/>
    </div>
</div>
  );
}export default App;
